## 📖 Introduction
The **Dengxian Skin Pack** is a specialized visual expansion for the "Dengxian" ecosystem. This mod focuses on high-performance assets and immersive aesthetics to bring a true "Cultivation" atmosphere to your game.

## ✨ Key Features
* 🎭 **Unlock Exclusive Skins:** Access a wide range of custom skins for **Characters**, **Items**, and **Structures**.
* 🎨 **Custom UI:** Fully redesigned crafting icons and menus tailored for the Xianxia theme.
* ⚡ **High Optimization:** High-quality `.tex` files (512x1024) optimized to prevent memory leaks and lag.
* 🌐 **Full Localization:** Complete string support for seamless integration.

## 🚀 Installation Guide

### 1️⃣ Step 1: Download
<p align="center">
  Click the <b>Download</b> button at the top or visit the <a href="https://github.com/lTMeol/DST-ModSkin-Dengxian/releases">Releases</a> page.

### 2️⃣ Step 2: Extract to Mods Folder
Extract the downloaded folder to the following directory:
`Documents\Klei\DoNotStarveTogether\mods\`

<p align="center">
  <img src="https://github.com/user-attachments/assets/2dbf81b2-7100-44ed-89ff-c8b8a686869d" width="600">
  <br>
  <i>Ensure the mod folder containing <b>modinfo.lua</b> is placed directly inside the "mods" directory.</i>
</p>

### 3️⃣ Step 3: Enable in Game
<p align="center">
  <img src="https://github.com/user-attachments/assets/167e5b43-c5a8-42fe-941e-51d67a0c7c3f" width="600">
  <br>
  Launch the game -> Go to <b>Mods</b> -> Find <b>【登仙】 Mod Skin Pack</b> and toggle it to <b>Enabled</b>.
</p>

## ✨ Tính năng chính
* 🎭 **Skin độc quyền:** Mở khóa các trang phục tùy chỉnh cho **Nhân vật**, **Vật phẩm**, và **Công trình**.
* 🎨 **Giao diện (UI) riêng:** Làm lại toàn bộ các biểu tượng chế tạo và menu theo phong cách Tu Tiên.
* ⚡ **Tối ưu hóa cao:** Sử dụng các tệp `.tex` chất lượng cao (512x1024) được tối ưu để tránh lỗi tràn bộ nhớ (memory leak).
* 🌐 **Việt hóa hoàn toàn:** Hỗ trợ ngôn ngữ tiếng Việt hoàn chỉnh cho mọi thành phần.
