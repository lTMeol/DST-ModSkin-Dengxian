## 📖 Introduction
The **Dengxian Skin Pack** is a specialized visual expansion for the "Dengxian" ecosystem. This mod focuses on high-performance assets and immersive aesthetics to bring a true "Cultivation" atmosphere to your game.

---

## 🚀 Installation Guide

### 1️⃣ Step 1: Select Version & Download
After clicking the **Download** button above, you will be redirected to the **Releases** page. Choose the latest version and download the `.zip` file under the **Assets** section.

<p align="center">
  <img src="https://github.com/user-attachments/assets/fcf786b5-0954-4f14-9d37-e2a57ec01746" width="650">
  <br>
  <i>Make sure to select the correct .zip file for your desired version.</i>
</p>

### 2️⃣ Step 2: Extract to Mods Folder
Extract the downloaded folder to the following directory:
`Steam\steamapps\common\Don't Starve Together\mods`

<p align="center">
  <img src="https://github.com/user-attachments/assets/2dbf81b2-7100-44ed-89ff-c8b8a686869d" width="600">
  <br>
  <i>Ensure the mod folder containing <b>modinfo.lua</b> is placed directly inside the "mods" directory.</i>
</p>

### 3️⃣ Step 3: Enable in Game
Launch the game -> Go to **Mods** -> Find **【登仙】 Mod Skin Pack** and toggle it to **Enabled**.

  <img src="https://github.com/user-attachments/assets/036d7073-5431-46e9-bf1b-3d1bdd06bdc8" width="600">
</p>
