## 🚀 Installation Guide

Follow these 3 simple steps to get started:

### 1️⃣ Step 1: Download
Click the **Download** button at the top or visit the [Releases](https://github.com/lTMeol/DST-ModSkin-Dengxian/releases) page to get the latest `.zip` file.

### 2️⃣ Step 2: Extract to Mods Folder
Extract the downloaded folder to the following directory:
`Documents\Klei\DoNotStarveTogether\mods\`

> <img src="https://github.com/user-attachments/assets/2dbf81b2-7100-44ed-89ff-c8b8a686869d" width="600">  
> *Ensure the mod folder containing **modinfo.lua** is placed directly inside the "mods" directory.*

### 3️⃣ Step 3: Enable in Game
Launch the game -> Go to **Mods** -> Find **【登仙】 Mod Skin Pack** and toggle it to **Enabled**.

> <img src="https://github.com/user-attachments/assets/167e5b43-c5a8-42fe-941e-51d67a0c7c3f" width="600">
