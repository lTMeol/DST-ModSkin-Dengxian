# 🌀 【登仙】 (Dengxian) - Mod Skin Pack
**A premium skin system for characters in the "Dengxian" (Đăng Tiên) universe.**
*An extension project for the Dengxian Mod ecosystem in Don't Starve Together.*

---

## 📖 Introduction
This mod provides high-quality visual skins and UI enhancements for the **Dengxian** mod. It features custom assets, optimized `.tex` files, and a dedicated UI style to bring a true "Cultivation" (Xianxia) experience to the game.

## ✨ Key Features
* **Unique Skins:** Exclusive designs for characters like **Han Thien Ton**, **Tu Van Ma Quan**, **Tam Tieu Nuong Nuong**, and more.
* **Custom UI:** Completely redesigned crafting icons for items and structures.
* **Optimization:** High-performance assets (512x1024) optimized to prevent memory leaks and lag.
* **Localization:** Accurate string support for character names and descriptions.

## 🚀 Installation Guide
1. Download the latest `.zip` file from the [Releases](https://github.com/lTMeol/DST-ModSkin-Dengxian/releases) section.
2. Extract the folder to: `Documents/Klei/DoNotStarveTogether/mods/`.
3. Enable the mod in the **Mods** menu within the game.
